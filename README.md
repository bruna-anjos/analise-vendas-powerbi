# Dashboard de Análise de Vendas

## Objetivo

Projeto desenvolvido no Power BI para análise de vendas, faturamento, clientes e devoluções.

## Indicadores Monitorados

- Faturamento Total
- Quantidade Vendida
- Ticket Médio
- Crescimento Anual
- Devoluções
- Análise por Produto
- Análise por Região

## Ferramentas Utilizadas

- Power BI
- Power Query
- DAX
- Excel
